# Tic Tac Toe AI using Minimax Algorithm

## Description
This project implements an AI-powered Tic Tac Toe game using the Minimax algorithm. 
The AI evaluates all possible game states and selects the optimal move, making it unbeatable.

## Features
- Human vs AI gameplay
- Minimax algorithm implementation
- GUI using Tkinter
- Win / Draw detection
- Restart functionality

## Technologies Used
- Python
- Tkinter
- Minimax Algorithm (Adversarial Search)

## How to Run
1. Install Python
2. Run:
   python tictactoe_ai.py
